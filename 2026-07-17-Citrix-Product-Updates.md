# Citrix Product Updates — 2026-07-17

Summary of "What's New" documentation updates published in the last 24 hours (2026-07-16 to 2026-07-17) across [docs.citrix.com](https://docs.citrix.com), [docs.netscaler.com](https://docs.netscaler.com), and [docs.xenserver.com](https://docs.xenserver.com).

## Citrix

### Citrix DaaS

Source: [Citrix DaaS What's New](https://docs.citrix.com/en-us/citrix-daas/whats-new.html)

- Citrix Advisor now includes new environment checks that provide actionable recommendations to help improve reliability, performance, and operational excellence.
- Configuration log search in Web Studio's Logging section can now be extended to include low-level operation details (e.g., machine names, VDI identifiers).
- Autoscale capacity buffer calculation now excludes machines that are draining due to scheduled restarts.
- Citrix Aidrien responses now include deep links to the corresponding DaaS console view with search filters applied automatically.
- You can now connect an AWS account to Web Studio without configuring Bring Your Own License (BYOL) at the same time.
- Finding Active Directory domains and organizational units (OUs) is now faster in Web Studio via a new search capability in the Location/domain dropdown.
- Task Center now includes a "Recurring" tab showing tasks scheduled to run periodically or in the future, alongside the existing Recent tasks tab.
