## XenServer

### Conversion Manager (version 2026.1.0, released July 6, 2026)

Sources:
- [XenCenter — Conversion Manager What's New](https://docs.xenserver.com/en-us/xencenter/current-release/conversion-manager/whats-new.html)
- [XenServer 8 — Conversion Manager What's New](https://docs.xenserver.com/en-us/xenserver/8/conversion-manager/whats-new.html)
- [XenServer 9 — Conversion Manager What's New](https://docs.xenserver.com/en-us/xenserver/9/conversion-manager/whats-new.html)

- Support for converting additional Linux guests: Red Hat Enterprise Linux 9, Rocky Linux 8 and 9, and Ubuntu 22.04 and 24.04.
- Support for converting Linux guests using the XFS file system or UEFI boot mode.
- Support for converting VMs from VMware 9 to XenServer.
- Support for converting VMs to XenServer 9 as a destination.
- Support for converting a single virtual disk up to 16 TiB on GFS2 storage repositories.
- New XenCenter option to automatically start each VM after its conversion completes.
- Preview support for converting Windows Server 2025 guests.
